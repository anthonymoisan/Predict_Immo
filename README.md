# Predict_Immo

L'objectif de ce projet de Datamining est de préduire les prix à Paris à partir d'un certain nombre de variables explicatives :
* surface
* commune
* ...

L'objectif est aussi de visualiser les résultats à travers Power BI et de voir des interactions avec du code Python

## Pour démarrer

Il suffit de cloner ou copier en local le répetoire.

L'organisation du répertoire est la suivante : 
* src : un Jupyter Notebook, les classes pythons pour avoir une version du modèle de prédictions, l'intégration dans Power BI du modèle de machine learning 
* input : le jeu de données est trop lourd mais vous pouvez télécharger les données 2021 et 2022 sous https://www.data.gouv.fr/fr/datasets/demandes-de-valeurs-foncieres-geolocalisees/ 
* documentation.pdf : le fichier notice descriptive qui explique les données du problème, PredictImmo l'analyse exploratoire et le modèle retenu, Projet_Immo_PowerBI l'export Power BI
